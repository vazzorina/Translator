# Translator
Фоновая утилита на C++ для перевода выделенного текста с английского на русский по горячим клавишам с использованием API Yandex.

### 🛠 Сборка и запуск
Требования: 
- Qt 5.15+ / 6.x.
- Git (для клонирования репозитория)
  
Запуск программы: 
1. Откройте папку для проекта в терминале и клонируйте репозиторий с помощью следующей команды:
```text
git clone https://github.com/vazzorina/Translator
```
2. Через Qt Creator откройте файл Translate.pro. Выберите нужный компилятор и конфигурацию.
3. В файле `yandextranslete.h` вставьте API-ключ Яндекс-Переводчика `API-key-translate-Yandex` в строке:
```text
const QString apiKey = "API-key-translate-Yandex";
```
> [!IMPORTANT]
> Утилита изначально была разработана для работы с API-ключом от Яндекс-Переводчика. Без этого ключа утилита работать не будет.
4. Нажмите **Build** - для сборки проекта
5. Нажмите **Run** - для запуска проекта

---

### Как использовать?
Инструкция:
1. Выделите текст на английском.
2. Нажмите сочетание клавиш `Ctrl + ~` (клавиша Ё).
3. В случае, если нет ошибок, отобразится окно с переводом на русский.
> [!NOTE]
> Утилита работает в фоновом режиме, у нее нет основного окна.
4. Чтобы закрыть программу, откройте системный трей, нажмите на иконку правой кнопки мыши, чтобы открылось контекстное меню. Нажмите выход и утилита прекратит работу.

> [!NOTE]
> Перевод текста основан на копировании выделенного текста в буфер обмена и отправка его по API-ключу в переводчик. Если в документах или других файлах, которые вы хотите перевести, включен запрет на копирование, то выделенный текст не скопируется и перевод не поступит.
